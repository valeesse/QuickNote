为效率而生的核心功能

不是另一个笔记应用，而是你桌面角落里随时待命的效率搭档。

实时 Markdown 渲染

基于 Tiptap 编辑器，输入即渲染。支持任务列表、代码块、图片粘贴，无需切换预览模式。

剪贴板历史管理

自动捕获剪贴板内容，支持搜索、置顶、去重。一键将剪贴板内容转为便签，再也不怕丢失复制的内容。

全局快捷键 & 系统托盘

Ctrl+Alt+N 全局新建便签，弹窗失焦自动隐藏。常驻系统托盘，用完即走，零干扰。

双模式同步

WebDAV 本地同步或云端 PostgreSQL 同步，两种模式互斥。因果版本向量处理冲突，数据安全不丢失。

版本历史 & 回收站

每次保存自动生成版本快照，支持回滚和置顶。便签软删除，误删可从回收站恢复。

端到端加密

AES-256-GCM 加密存储凭据，同步通道安全隔离。你的笔记内容只属于你。

快速开始

1

克隆仓库

$ git clone https://github.com/valeesse/QuickNote.git
2

启动桌面端

$ npm --prefix apps/desktop install $ npm run tauri -- dev

环境要求：Node.js 20+、rust 1.96+

3

或部署云端 Web

$ cp .env.example .env # 编辑配置 $ docker compose up -d --build

需要 Docker Compose v2，默认入口 http://localhost:8081

常见问题

QuickNote 支持哪些操作系统？

桌面端目前支持 Windows。基于 Tauri 2 构建，未来可扩展到 macOS 和 Linux。Web 端可在任何现代浏览器中访问。

我的数据安全吗？

QuickNote 采用"本地优先"策略，数据默认存储在你的电脑 SQLite 数据库中。WebDAV 同步使用你自己的存储服务（如 NAS），云端模式支持 Docker 自托管。凭据使用 AES-256-GCM 加密，你完全掌控自己的数据。

WebDAV 同步支持哪些服务？

任何标准 WebDAV 协议的服务都可以，包括：坚果云、InfiniCLOUD、fnOS NAS、群晖 WebDAV Server、Nextcloud 等。只需在设置中配置 WebDAV 地址、用户名和密码即可。

QuickNote 和 Notion / Obsidian 有什么区别？

QuickNote 不追求知识库、双链、数据库等重型功能，专注于"便签速记 + 剪贴板管理"这个细分场景。启动更快、占用更少、用完即走。两者并不冲突，很多用户同时使用 QuickNote 和 Obsidian。

QuickNote 是 Apache 2.0 开源项目，欢迎在 GitHub Issues 提交 Bug 报告和功能建议，也欢迎提交 Pull Request。
